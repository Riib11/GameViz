# GameViz


## Prerequisites

- pip3
- git
- dot

## Installation

    # downloads this repository
    git clone https://github.com/Riib11/GameViz.git ~/git/GameViz
    # installs bash commands, installs python3 module
    sh ~/git/GameViz/install.sh

## Updating

    sh ~/git/GameViz/update.sh

## Compiling to HTML

    gdot <input>.gdot
